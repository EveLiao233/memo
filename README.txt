最初版本——DT (Time Tracer)

    来源于Android 应用开发课程大作业，主要目标为通过进度条明确展示事项进度。

开发人员：
廖谊 王宝亿 牛晶晶

----------------------------------------------------------------
现版本——Memo (memograph)

    我们新增了两名成员，并且在DT的基础上进行了一定的修改和完善，使其功能更加丰富，界面更加美观。将DT升级为备忘录，将静态提醒升级为后台提醒，更加实用。并且添加了用户登录、注册、修改头像等功能，还可以将本地数据同步到服务器端以及从服务器端将数据同步到一台新的手机上。

可执行文件

memo_SC.apk为包含用户登录、注册、数据上传服务器功能的安装包

DT_v3.0.apk为在DT基础上迭代的第三版本，即memo上一版本，未包含用户登录、注册与访问服务器功能的安装包，同时部分界面尚未更新。

由于服务器为组员自行搭建，不会时刻开启，并且该APP需要注册后才可使用。因此提供DT_v3.0版本供使用和测试。
